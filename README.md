# 🚀 풀스택 연습 프로젝트

프론트엔드와 백엔드 연결 학습을 위한 간단한 웹 애플리케이션입니다.

## 📁 프로젝트 구조

```
full_stack/
├── backend/          # 백엔드 (Node.js + Express)
│   ├── package.json
│   └── server.js
├── frontend/         # 프론트엔드 (HTML + CSS + JavaScript)
│   ├── index.html
│   ├── style.css
│   └── script.js
└── README.md
```

## 🔧 설치 및 실행 방법

### 1. 백엔드 서버 실행

```bash
# 백엔드 폴더로 이동
cd backend

# Node.js 패키지 설치
npm install

# 서버 실행
npm start
```

서버가 성공적으로 실행되면 `http://localhost:3000`에서 백엔드 API가 동작합니다.

### 2. 프론트엔드 실행

프론트엔드는 별도의 설치 과정이 필요없습니다. 웹 브라우저에서 `frontend/index.html` 파일을 열면 됩니다.

**방법 1: 파일 직접 열기**
- `frontend/index.html` 파일을 더블클릭하여 브라우저에서 열기

**방법 2: VS Code Live Server 사용 (권장)**
- VS Code에서 `frontend/index.html` 파일을 열고
- 마우스 우클릭 → "Open with Live Server" 선택

## 🎯 기능 설명

### 1. 서버 상태 확인
- **목적**: 백엔드 서버가 정상적으로 동작하는지 확인
- **API**: `GET /`
- **사용법**: "서버 상태 확인" 버튼 클릭

### 2. 로그인 기능
- **목적**: 사용자 인증 처리
- **API**: `POST /api/login`
- **테스트 계정**:
  - `admin` / `1234`
  - `user1` / `pass1`
  - `user2` / `pass2`

### 3. 데이터 처리
- **목적**: 백엔드에서 데이터를 처리하고 결과 반환
- **API**: `POST /api/process-data`
- **기능**:
  - **2배 계산**: 숫자를 입력하면 2배로 계산
  - **문자 역순**: 문자열을 역순으로 변경
  - **그대로 반환**: 입력값을 그대로 반환

### 4. 사용자 목록 조회
- **목적**: 등록된 모든 사용자 정보 조회
- **API**: `GET /api/users`
- **사용법**: "사용자 목록 불러오기" 버튼 클릭

## 🔄 데이터 흐름

```
1. 프론트엔드 (브라우저)
   ↓ HTTP 요청 (fetch API)
2. 백엔드 (Express 서버)
   ↓ 데이터 처리
3. 백엔드에서 JSON 응답
   ↓ HTTP 응답
4. 프론트엔드에서 결과 표시
```

## 🧪 테스트 시나리오

4. 숨겨진 섹션들이 나타나는지 확인

### 시나리오 3: 데이터 처리 테스트

## 🛠️ 주요 기술

### 백엔드
- **Node.js**: JavaScript 런타임
- **Express**: 웹 프레임워크
- **CORS**: 교차 출처 리소스 공유 허용
- **body-parser**: JSON 요청 본문 파싱

### 프론트엔드
- **HTML5**: 마크업
- **CSS3**: 스타일링 (그라디언트, 애니메이션)
- **JavaScript ES6+**: 로직 처리
- **Fetch API**: HTTP 통신

## 🚨 문제 해결

### 1. 서버 연결 오류
**문제**: "서버와 연결할 수 없습니다" 메시지
**해결책**:
- 백엔드 서버가 실행 중인지 확인
- `http://localhost:3000`에 직접 접속해보기
- 방화벽 설정 확인

### 2. CORS 오류
**문제**: 브라우저 콘솔에서 CORS 에러
**해결책**:
- 백엔드 서버에 CORS 미들웨어가 설정되어 있는지 확인
- 서버 재시작

### 3. 로그인 실패
**문제**: 올바른 계정 정보 입력했지만 로그인 실패
**해결책**:
- 네트워크 탭에서 API 요청/응답 확인
- 백엔드 콘솔 로그 확인
